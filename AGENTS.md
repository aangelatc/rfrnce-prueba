# AGENTS

## Objetivo del proyecto
Crear un sitio web profesional basado en el documento `prd-rfrnce-sitio-web.md`.

## Roles y agentes

- **Project Manager (PM)**
  - Coordina la ejecución del proyecto.
  - Estructura los entregables, define prioridades y revisa los avances.

- **Designer / UX Specialist**
  - Define la experiencia de usuario y la apariencia visual.
  - Crea wireframes, prototipos e identifica los requisitos de interfaz.

- **Front-end Developer**
  - Implementa la interfaz web con HTML, CSS y JavaScript.
  - Asegura responsividad, accesibilidad y compatibilidad.

- **Content Specialist**
  - Revisa y adapta el contenido del `prd-rfrnce-sitio-web.md`.
  - Organiza textos, secciones y llamados a la acción.

- **QA / Tester**
  - Comprueba que la funcionalidad y el diseño cumplan los requisitos.
  - Realiza pruebas en distintos navegadores y dispositivos.

## SKILLS necesarias

- Análisis de requisitos
  - Interpretar el documento `prd-rfrnce-sitio-web.md`.
  - Extraer secciones, objetivos y público objetivo.

- Arquitectura de información
  - Definir la estructura del sitio.
  - Establecer la navegación y jerarquía de contenido.

- Diseño visual y UX
  - Crear estilo visual coherente.
  - Elegir paleta de colores, tipografías y componentes UI.

- HTML/CSS avanzado
  - Implementar layouts modernos.
  - Usar prácticas de CSS para responsividad.

- JavaScript / Interactividad
  - Añadir comportamientos dinámicos básicos.
  - Validaciones, menús móviles, carruseles o efectos ligeros.

- Gestión de proyecto ligera
  - Mantener un plan de trabajo claro.
  - Actualizar estado del proyecto y prioridades.

- Control de versiones (opcional)
  - Usar Git para seguimiento de cambios.
  - Registrar commits claros y organizar ramas.

## Recursos recomendados

- Documentación de HTML y CSS.
- Guías de accesibilidad web.
- Plantillas o frameworks ligeros, si se decide usar alguno.
- Herramientas de prototipado y revisión.
